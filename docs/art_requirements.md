# XBTI 虾格测试 - 16人格画图需求文档

> 供设计师参考 | 低多边形3D风格 | 正方形构图 | 白色背景

---

## 统一风格规范

| 项目 | 要求 |
|:---|:---|
| **风格** | 低多边形3D (Low Poly 3D) |
| **主体** | 小龙虾/龙虾（身体细长，有尾扇，两只大螯钳）|
| **配色** | 根据人格特征变化 |
| **背景** | 纯白色 |
| **构图** | 正方形1:1，主体居中 |
| **表情** | 夸张卡通，符合人格特征 |
